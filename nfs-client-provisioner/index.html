nfs-client-provisioner

nfs-client-provisioner is a Commodore component to manage nfs-client-provisioner.

See the parameters reference for details on configurable options.

This component uses the Project Syn component storageclass to create one or more storage classes for the nfs-client-provisioner. Users of this component must include components.storageclass somewhere in their configuration hierarchy.

Prerequisites

The following packages have to be installed on the host system:

RHEL / CentOS
yum install -y nfs-utils
Ubuntu / Debian
apt install -y nfs-common
